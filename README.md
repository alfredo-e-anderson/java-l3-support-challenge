# Java L3 Support - Challenge Técnico (Payment API)

¡Bienvenido/a al proceso de evaluación técnica para el rol de **Ingeniero de Soporte Avanzado / Desarrollador Java L3**! 

El objetivo de este ejercicio es evaluar tus habilidades analíticas, capacidad para diagnosticar problemas en entornos productivos, resolución de bugs de precisión y tus conocimientos de desarrollo sobre el ecosistema Java y Spring Boot.

---

## 📌 Contexto del Proyecto

Te encuentras con `payment-api`, un microservicio central de procesamiento de transacciones financieras. El sistema utiliza **Java**, **Spring Boot**, **Spring Data JPA** y una base de datos en memoria **H2** para facilitar su ejecución inmediata.

Recientemente, el equipo de operaciones y negocio ha reportado una serie de incidentes en producción, así como un nuevo requerimiento urgente que debe ser implementado con la mayor precisión posible.

---

## 🚀 Requisitos Previos y Ejecución

Para levantar y trabajar en este proyecto necesitarás:
* **Java 17** (o superior).
* **Maven 3.8+**.
* Tu IDE de preferencia (IntelliJ, Eclipse, VS Code, etc.).

### Ejecución de la aplicación:
Puedes clonar el repositorio y correr el proyecto usando tu IDE o mediante la terminal con el siguiente comando:
